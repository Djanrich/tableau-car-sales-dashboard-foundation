# Tableau Car Sales Dashboard (Foundational Project)

## Overview
This project represents an early Tableau dashboard built to practice core data visualization
and dashboard design concepts. The dashboard analyzes year-to-date car sales performance
across different dimensions such as body style, color, dealer region, and manufacturer.

## Business Context
The dashboard simulates a car dealership environment where leadership needs visibility into
sales performance trends, pricing behavior, and vehicle sales volume to support data-driven
decision-making.

## Key KPIs
- Year-to-Date (YTD) Total Sales
- Year-over-Year (YOY) Growth in Total Sales
- YTD Average Price
- YOY Growth in Average Price
- YTD Cars Sold
- YOY Growth in Cars Sold

## Visualizations
- Weekly YTD sales trend
- YTD total sales by body style
- YTD total sales by color
- YTD cars sold by dealer region
- Company-wise sales performance table

## Tools Used
- Tableau Public
- Microsoft Excel

## Dataset
The dataset used in this project is a publicly available car sales dataset used for learning
and practice purposes.
